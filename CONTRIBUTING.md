# Contributing

Thanks for helping improve `@arclogi/nepali-date-picker`.

## Local development

```sh
npm install
npm run lint
npm run typecheck
npm run test
npm run build
```

## Project standards

- Keep the public API TypeScript-first and framework-friendly.
- Add tests for date conversion, date math, and UI behavior when changing those areas.
- Keep component styles scoped with the `ndp-` class prefix.
- Document user-facing API changes in `README.md`.

## Commit style

Use short, clear commit messages that explain the project change:

```txt
Add calendar grid utilities
Improve date input keyboard handling
```
